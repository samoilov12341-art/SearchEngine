# Search Engine
Цель проекта: индексировать сайты и страницы на них, находить леммы и предоставлять эффективный поиск по
индексированным данным. Поиск поддерживает релевантность и отображает сниппеты with выделенным жирным
шрифтом искомых слов.

Особенности
Индексация страниц с поддержкой лемматизации (морфология) только для русского языка через lucene morphology
Поиск по одному сайту или по всем индексированным сайтам
Релевантность и полнотекстовый поиск
API и веб-интерфейс для управления индексами и поиск
Управление индексами через UI: dashboard, management, search
Dashboard: статистика по индексируемым сайтам и статусу индексации
Management: добавить новый сайт для индексирования или обновить уже проиндексированный
Search: поле поиска, кнопка "найти" возвращает список страниц со сниппетами и выделением искомых слов жирным шрифтом
Настройка осуществляется через application.yaml
В свойстве indexing-settings.sites необходимо указать сайты по которым будет производиться индексация
Перед запуском программы локально, необходимо установить базу данных MySQL, а так же указать в application.yaml
логин, пароль и url для подключения к ней

Технологии
Версия Java 20.0.2
Spring Framework + JPA
Apache Lucene (с поддержкой морфологии через Lucene Morphology)
Jsoup для парсинга веб-страниц
База данных: MySQL
